# 5G Architecture — Network Simulation

A research and simulation project exploring 5G network architecture,
implemented in Python via Jupyter Notebook. Includes a full presentation
of the 5G stack and a simulation/modeling component.

##  Overview

The project covers the architecture and simulation of a 5G network,
analyzing key components such as RAN, Core Network, and slicing concepts
through code and visual presentation.

##  Files

| File | Description |
|------|-------------|
| `5gProjectCode.ipynb` | 5G network simulation & modeling in Python |
| `5gPresentation.pptx` | Architecture overview & findings presentation |
| `Link For Code Colab.txt` | Google Colab link to run the notebook online |

##  How to Run

**Option 1 — Google Colab (recommended):**
Open the link inside `Link For Code Colab.txt`

**Option 2 — Local Jupyter:**
```bash
pip install jupyter
jupyter notebook 5gProjectCode.ipynb
```
